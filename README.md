# JAVA2
## 1일차 - Github용어
* Repository(레포지토리)
  - 디렉터리 또는 폴더를 한곳에 모아두는곳
* Branch(브런치)
  - 독립적인 저장공간을 제공하여 여러 개발자들이
    기능이나 버그 등 서로 다른 작업을 병렬적으로 수행가능
* Default Branch(기본 브런치)
  - 저장소를 사용할 디렉토리를 초기화 할때 사용되는
    브런치로 Master나 Main을 사용함.
* local Repository
  - 개발자의 PC에 있는 디렉토리
* remote Repository
  - 서버에 있는 디렉토리
